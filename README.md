🛒 JavaScript Shopping Cart System

This project is a Shopping Cart System built using JavaScript as part of my 30 Days JavaScript Challenge (Day 29).
The application allows users to add products to the cart, update product quantities, remove items, and calculate the total price dynamically. The cart data is stored using Local Storage, so items remain saved even after refreshing the page.


✨ Features
🛍 Display products dynamically
➕ Add products to cart
🔼 Increase product quantity
🔽 Decrease product quantity
❌ Remove items from cart
💰 Automatic total price calculation
💾 Cart saved in Local Storage
🔄 Cart persists after page refresh

🛠 Technologies Used
HTML
CSS
JavaScript
Local Storage API

📂 Project Functionality
Product Display
Products are dynamically rendered using JavaScript arrays.
Add to Cart
Users can click Add to Cart to store items in the cart.
Quantity Management
Users can increase or decrease product quantities directly from the cart.
Remove Item
Products can be removed completely from the cart.
Local Storage
Cart data is saved using localStorage, ensuring items stay in the cart even after refreshing the page.
